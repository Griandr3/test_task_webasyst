# Плагин "Подарок для вас" для Shop-Script (Webasyst)

## Описание

Плагин добавляет на сайт интерактивную страницу с розыгрышем подарков, где посетители могут получить случайный товар из списка, настроенного администратором. После выбора товара пользователь может указать свой email и получить письмо с информацией о подарке.

## Функциональность

### Административная часть (Backend)
- **Настройка товаров**: Администратор может указать ID товаров через запятую, которые будут участвовать в акции
- Настройки сохраняются стандартным способом Webasyst
- Доступ через: `Магазин → Плагины → testplugin → Настройки`

### Публичная часть (Frontend)
1. **Страница розыгрыша**:
   - Заголовок "Подарок для вас!"
   - Кнопка "Крутить" для запуска розыгрыша
   - AJAX-запрос выбирает случайный товар из настроенного списка

2. **Карточка товара**:
   - Название товара
   - Изображение (150x150px)
   - Цена с форматированием

3. **Форма отправки email**:
   - Поле для ввода email с валидацией
   - Кнопка "Получить письмо"
   - Отправка информации о товаре на указанный email

### Email-уведомление
Письмо содержит:
- Тема: "Ваш подарок!"
- Название товара
- Цена товара
- Прямую ссылку на товар

## Установка

1. Скопируйте папку `testplugin` в `wa-apps/shop/plugins/`
2. Зайдите в админку Webasyst: `Магазин → Плагины`
3. Найдите плагин "testplugin" и активируйте его
4. Перейдите в настройки плагина
5. Укажите ID товаров через запятую (например: `1,5,10,15`)
6. Сохраните настройки
